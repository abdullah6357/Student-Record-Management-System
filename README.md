#Overview

The Student Record Management System is a simple yet practical Python project that lets you manage student information using a CSV file as a lightweight database.
It features a clean, menu-driven interface where you can easily add, view, search, update, and delete student records. This project is perfect for practicing core Python skills like file handling, working with CSV data, functions, loops, and building interactive console applications.


#Project Objectives

Store and manage student data persistently in a CSV file
Implement full CRUD (Create, Read, Update, Delete) operations
Build a user-friendly command-line application
Practice real-world file handling and data management in Python

#Features

Add new student records
View all student records
Search students by ID or name
Update existing student information
Delete student records
Data automatically saved to CSV file
Easy-to-use menu interface
Clean and well-organized code

#Technologies Used

Python: programming language 
CSV Module: reading & writing data
Jupyter Notebook: Development & testing
