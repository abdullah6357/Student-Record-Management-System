## 📌 Overview

The **Student Record Management System** is a simple yet practical Python project that lets you manage student information using a CSV file as a lightweight database.

It features a clean, menu-driven interface where you can easily **add, view, search, update, and delete** student records. This project is perfect for practicing core Python skills like file handling, working with CSV data, functions, loops, and building interactive console applications.

---

## Features

 Add new student records
 View all student records
 Search students by ID or Name
 Update existing student information
 Delete student records
 Automatic data saving in CSV format
 Clean menu-driven interface

## 💻 Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/abdullah6357/Student_CSV.git
cd Student_CSV
```

## 2. Install Jupyter Notebook
```
pip install notebook
```

## 3. Launch Jupyter Notebook
```
jupyter notebook
```
Then open the file Student_CSV (1).ipynb
